# BlackJack-Humilde---Ibai-Manso---Samuel-Sainz-Terrones

Este es un proyecto simple que hemos hecho para nuestro primer de reto de las asignaturas de 
entorno cliente e interfaces de nuestro segundo año de grado superior de DAW (Desarrollo de aplicaciones Web),
nuestro proyecto trata de un juego de Blackjack, desarrollado principalmente en JavaScript, HTML y CSS.

El juego de Blackjack permite al usuario jugar contra la "banca" en el clásico formato de este popular juego de cartas. El objetivo es acercarse lo más posible a 21 puntos sin pasarse.

--Requisitos previos--
Para que el juego funcione correctamente, asegúrate de tener instalado un servidor local. Las imágenes de las cartas no se mostrarán adecuadamente si solo abres el archivo HTML en el navegador. Esto es debido a restricciones de seguridad de los navegadores al cargar imágenes locales sin un servidor.

Soluciones:
Usar el plugin de VS Code "Live Server": Puedes abrir el proyecto directamente en Visual Studio Code y luego iniciar "Live Server" para previsualizar el juego sin problemas.
Subir a un servidor: Si cuentas con un servidor en la nube o local, puedes subir el proyecto para visualizarlo correctamente en cualquier navegador.

--Cómo jugar--
Descarga el repositorio o clona el proyecto en tu máquina.
Abre la carpeta en Visual Studio Code.
Inicia "Live Server" desde VS Code o despliega el proyecto en un servidor.
Abre el juego en tu navegador y sigue las instrucciones para jugar.
